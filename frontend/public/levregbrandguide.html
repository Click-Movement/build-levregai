06 — Usage Guidelines

Do's & Don'ts

These rules apply to both modes. Consistent application ensures the LevReg.AI brand remains coherent, accessible, and professional across all touchpoints.

✓
Do
  • Use #14B870 exclusively for the logo, primary CTA buttons, key highlights, and important icons in both modes.
  • In dark mode, use #060711 / #05070A as backgrounds and #F9FAFB for primary text.
  • In light mode, use #F9FAFB / #FFFFFF as backgrounds and #0D1117 for primary text.
  • Darken the accent blue to #0284C7 in light mode to maintain AA contrast on white surfaces.
  • Use semantic state colors strictly for system feedback — their meaning must remain consistent across modes.
  • Use CSS design tokens (--green, --text-pri, etc.) in all code so both modes work from a single stylesheet.
  • Always use LevReg_WhiteBlue.png on dark backgrounds and LevReg_BlackBlue.png on light backgrounds.
✕
Don't
  • Do not use #14B870 as a large background fill in either mode — it is a high-signal accent only.
  • Do not use the light cyan #0EA5E9 as a link color in light mode — it fails AA contrast on white; use #0284C7 instead.
  • Do not mix dark-mode surfaces (#111827) into a light-mode layout, or vice versa.
  • Do not use warning amber or error red for decorative purposes in either mode.
  • Do not introduce new colors outside this palette without formal brand approval.
  • Do not use pure black (#000000) in light mode — use #0D1117 for warmth and visual comfort.
  • Do not place the white logo on white/light backgrounds, or the black logo on dark backgrounds — both become illegible.
